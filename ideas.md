# JurisFinance — Ideias de Design

## Opção 1: Advocacia Clássica Modernizada
**Design Movement:** Corporate Brutalism com toques de Legaltech
**Core Principles:**
- Hierarquia tipográfica forte com serifa para títulos
- Dados sempre em destaque — números grandes, legíveis
- Paleta sóbria com acento dourado (remetendo à justiça)
**Color Philosophy:** Azul-marinho profundo (#1a2744) como base, dourado (#c9a84c) como acento, branco off-white (#f8f7f4) para fundos
**Layout Paradigm:** Sidebar fixa escura + área de conteúdo clara, com cards de métricas em grade assimétrica
**Signature Elements:** Linha dourada divisória, ícone de balança estilizado, tabelas com zebra-striping sutil
**Interaction Philosophy:** Transições suaves, hover com elevação de card
**Animation:** Entrada de números com counter animation, gráficos com fade-in progressivo
**Typography System:** Playfair Display (títulos) + Inter (dados/corpo)

## Opção 2: Fintech Minimalista Escura
**Design Movement:** Dark Mode Fintech Premium
**Core Principles:**
- Fundo escuro com cards em cinza-chumbo
- Dados financeiros com destaque em verde/vermelho
- Densidade de informação alta sem poluição visual
**Color Philosophy:** Fundo #0f1117, cards #1c1f2e, verde esmeralda #10b981 para positivo, vermelho coral para negativo
**Layout Paradigm:** Sidebar compacta com ícones + dashboard em grid de 12 colunas
**Signature Elements:** Bordas neon sutis nos cards ativos, sparklines inline, badges de status coloridos
**Interaction Philosophy:** Feedback imediato, tooltips ricos em gráficos
**Animation:** Skeleton loading, gráficos com animação de desenho
**Typography System:** Space Grotesk (títulos) + JetBrains Mono (valores monetários)

## Opção 3: Escritório Jurídico Contemporâneo
**Design Movement:** Editorial Legal — inspirado em revistas jurídicas premium
**Core Principles:**
- Tipografia editorial como elemento visual principal
- Espaço em branco generoso, layout respirado
- Cores institucionais com acento terroso
**Color Philosophy:** Branco puro + cinza-ardósia (#475569) + terracota (#c2714f) como acento quente
**Layout Paradigm:** Sidebar clara com bordas sutis, conteúdo em colunas editoriais
**Signature Elements:** Grandes números tipográficos como decoração, separadores de seção com linha fina
**Interaction Philosophy:** Elegante e deliberado — sem animações excessivas
**Animation:** Fade suave, transições de página com slide
**Typography System:** Fraunces (display/títulos) + DM Sans (corpo/dados)

---

## Design Escolhido: Opção 3 — Escritório Jurídico Contemporâneo

Filosofia aplicada:
- Sidebar clara com navegação limpa
- Tipografia Fraunces para títulos de seção e grandes números
- DM Sans para dados, tabelas e formulários
- Acento terracota para CTAs e destaques
- Espaço generoso entre seções
- Cards com sombra suave, sem bordas pesadas
- Gráficos com paleta coesa (terracota, ardósia, âmbar)
